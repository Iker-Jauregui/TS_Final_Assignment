Preface

Welcome to the course materials for Time Series, a comprehensive master’s level course covering modern techniques for analyzing and forecasting time-dependent data.

Course Overview

Time series data is ubiquitous in science, engineering, economics, and business. This course provides a rigorous yet practical introduction to the analysis and forecasting of time series data using Python.

The course is structured in two parts:

Part I: Foundations

  • Chapter 1 - Fundamentals of Time Series: Components (trend, seasonality, cycles), stationarity concepts, decomposition methods, and autocorrelation analysis.

  • Chapter 2 - Forecasting Principles and Methods: The forecasting workflow, benchmark methods, accuracy metrics, and validation strategies for temporal data.

Part II: Modeling Approaches

  • Chapter 3 - Statistical Models: ARIMA, SARIMA, and their extensions. Model identification, estimation, diagnostics, and forecasting.

  • Chapter 4 - Machine Learning for Time Series: Transforming time series to supervised learning, feature engineering, tree-based methods (Random Forest, XGBoost), Multi-layer perceptron, Prophet, and hybrid approaches.

  • Chapter 5 - Deep Learning for Time Series: Recurrent networks (LSTM, GRU), Temporal Convolutional Networks, Transformers, specialized architectures (N-BEATS, N-HiTS, TFT), and foundation models (TimeGPT).

Learning Objectives

By the end of this course, you will be able to:

  1. Analyze time series data to identify patterns, trends, and seasonality
  2. Apply appropriate transformations to achieve stationarity
  3. Build and evaluate statistical forecasting models (ARIMA/SARIMA)
  4. Engineer features for machine learning approaches to time series
  5. Implement modern deep learning architectures for forecasting
  6. Critically evaluate and compare different modeling approaches
  7. Select appropriate methods based on data characteristics and requirements

Prerequisites

This course assumes familiarity with:

  • Python programming: Intermediate level (pandas, numpy, matplotlib)
  • Statistics: Probability distributions, hypothesis testing, regression
  • Machine learning: Supervised learning concepts, cross-validation, bias-variance trade-off
  • Deep learning: Neural network fundamentals, backpropagation

Code Execution

The code in these materials is designed to be:

  • Reproducible: Random seeds are set where applicable
  • Educational: Clarity is prioritized over efficiency
  • Executable: You can run the code in your own environment
TipInteractive Learning

We encourage you to:

  1. Run the code examples yourself
  2. Modify parameters and observe the effects
  3. Apply techniques to your own datasets
  4. Complete the exercises before checking solutions

Pedagogical Approach

This course emphasizes:

  • Intuition before formalism: Visual and conceptual understanding precedes mathematical rigor
  • Learning by doing: Extensive hands-on coding exercises
  • Critical thinking: Evaluating model assumptions and limitations
  • Practical relevance: Real-world datasets and scenarios
ImportantA Note on Model Selection

There is no universally “best” forecasting method. The M-competitions have repeatedly shown that simple methods may often outperform complex ones. Throughout this course, we emphasize:

  • Starting with simple baselines
  • Using appropriate validation strategies
  • Letting the data guide model selection
  • Understanding the trade-offs between interpretability, accuracy, and computational cost
